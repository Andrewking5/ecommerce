# 02 - UI/UX 設計規範

## 風格定位

### 設計語言：Apple-inspired Minimalism
- **核心理念**：Less is More，專注於內容和功能
- **視覺層次**：清晰的信息架構，避免視覺噪音
- **用戶體驗**：直觀、流暢、無縫的操作體驗

## 使用者流程

### 註冊流程
```
訪客 → 點擊註冊 → 填寫表單 → 驗證郵件 → 登入成功 → 首頁
```

### 購物流程
```
首頁 → 瀏覽商品 → 商品詳情 → 加入購物車 → 查看購物車 → 結帳 → 支付 → 訂單確認
```

### 管理流程
```
管理員登入 → 儀表板 → 商品管理 → 訂單處理 → 用戶管理 → 數據分析
```

## 動畫設計 (Framer Motion)

### 頁面轉場
```javascript
// 頁面進入動畫
const pageVariants = {
  initial: { opacity: 0, y: 20 },
  in: { opacity: 1, y: 0 },
  out: { opacity: 0, y: -20 }
}

const pageTransition = {
  type: "tween",
  ease: "anticipate",
  duration: 0.4
}
```

### 按鈕互動
```javascript
// 按鈕懸停效果
const buttonVariants = {
```

## RWD 響應式規劃

### 斷點設定
```css
/* Mobile First 設計 */
--mobile: 320px    /* 手機 */
--tablet: 768px    /* 平板 */
--desktop: 1024px  /* 桌面 */
--wide: 1440px     /* 寬螢幕 */
```

### 佈局適配
- **手機 (320px-767px)**：
  - 單欄佈局
  - 漢堡選單
  - 大按鈕設計
  - 簡化導航

- **平板 (768px-1023px)**：
  - 兩欄佈局
  - 側邊欄導航
  - 適中按鈕
  - 網格商品展示

- **桌面 (1024px+)**：
  - 多欄佈局
  - 完整導航
  - 懸停效果
  - 豐富互動

### 圖片響應式
```css
/* 商品圖片適配 */
.product-image {
  width: 100%;
  height: auto;
  object-fit: cover;
}

@media (min-width: 768px) {
  .product-image {
    height: 300px;
  }
}

@media (min-width: 1024px) {
  .product-image {
    height: 400px;
  }
}
```

## 無障礙設計 (Accessibility)

### 色彩對比
- 文字與背景對比度 ≥ 4.5:1
- 重要按鈕對比度 ≥ 3:1

### 鍵盤導航
- Tab 鍵順序邏輯
- 焦點指示器明顯
- 快捷鍵支援

### 螢幕閱讀器
- 語義化 HTML 標籤
- Alt 文字描述
- ARIA 標籤支援

## 效能優化

### 圖片優化
- WebP 格式優先
- 延遲載入 (Lazy Loading)
- 響應式圖片 (srcset)

### 動畫效能
- 使用 transform 和 opacity
- 避免觸發重排
- 60fps 流暢動畫

### 載入策略
- 關鍵路徑優化
- 代碼分割
- 預載入重要資源
